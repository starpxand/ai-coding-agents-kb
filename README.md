## 🗂️ Классификация инструментов

### Standalone IDE
Отдельное приложение со встроенной средой разработки:
- **Cursor** — самый популярный AI-first редактор на базе VS Code
- **Windsurf** — IDE от Codeium с агентом Cascade
- **Google Antigravity** — agent-first IDE от Google DeepMind (ноябрь 2025)
- **Qoder** — агентная IDE от Alibaba с Quest Mode и Repo Wiki

### IDE Extension
Расширения/плагины для существующих сред (VS Code, JetBrains):
- **GitHub Copilot** — эталонный AI-ассистент от Microsoft/GitHub
- **Codeium** — бесплатная альтернатива Copilot
- **Tabnine** — AI-автодополнение с упором на приватность
- **Continue** — open-source расширение с поддержкой любых LLM

### CLI Agent
Инструменты командной строки, работающие в терминале:
- **OpenAI Codex** — агент OpenAI: CLI + облачный агент + IDE Extension
- **Claude Code** — агент Anthropic для терминала
- **Aider** — open-source CLI-агент с глубокой интеграцией Git
